# Transform date(s) to the format `yyyymmdd`

Transform date(s) to the format `yyyymmdd`

## Usage

``` r
to_yyyymmdd(x)
```

## Arguments

- x:

  A date or a vector of dates.

## Value

A vector of dates in the format `yyyymmdd.`

## Examples

``` r
if (FALSE) { # \dontrun{
to_yyyymmdd("2020-12-01")
to_yyyymmdd(c("2020-01-12", "1995-04-19"))
} # }
```

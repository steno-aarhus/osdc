# Create a vector of random ICD-8 diagnoses

ICD-8 is the 8th revision of the International Classification of
Diseases.

## Usage

``` r
create_fake_icd8(n)
```

## Arguments

- n:

  The number of ICD-8 diagnoses to generate.

## Value

A character vector of ICD-8 diagnoses.

## Examples

``` r
if (FALSE) { # \dontrun{
create_fake_icd8(1)
} # }
```

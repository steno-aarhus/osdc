# Transform date(s) to the format `yyww`

Transform date(s) to the format `yyww`

## Usage

``` r
to_yyww(x)
```

## Arguments

- x:

  A date or a vector of dates.

## Value

A vector of dates in the format `yyww`.

## Examples

``` r
if (FALSE) { # \dontrun{
to_yyww("2020-12-01")
to_yyww("2001-01-01")
to_yyww(c("2020-01-12", "1995-04-19"))
} # }
```

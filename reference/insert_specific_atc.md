# Insert specific ATC codes based on a proportion

Insert specific ATC codes based on a proportion

## Usage

``` r
insert_specific_atc(data, proportion = 0.3)
```

## Arguments

- data:

  A tibble.

- proportion:

  Proportion to be resampled. Defaults to 0.3.

## Value

A tibble with a proportion of resampled ATC codes for columns named
'atc'

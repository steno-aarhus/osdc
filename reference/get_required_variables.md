# Get a list of required variables from a specific register.

Get a list of required variables from a specific register.

## Usage

``` r
get_required_variables(register)
```

## Arguments

- register:

  The abbreviation of the register name. See list of abbreviations in
  [`get_register_abbrev()`](https://steno-aarhus.github.io/osdc/reference/get_register_abbrev.md).

## Value

A character vector of variable names.

## Examples

``` r
if (FALSE) { # \dontrun{
get_required_variables("bef")
} # }
```

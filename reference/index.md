# Package index

## Classifying diabetes

Main functions for classifying diabetes.

- [`algorithm()`](https://steno-aarhus.github.io/osdc/reference/algorithm.md)
  : A list of the algorithmic logic underlying osdc.
- [`classify_diabetes()`](https://steno-aarhus.github.io/osdc/reference/classify_diabetes.md)
  : Classify diabetes status using Danish registers.

## Documentation and examples

Functions used in the documentation and examples.

- [`registers()`](https://steno-aarhus.github.io/osdc/reference/registers.md)
  : Register variables (with descriptions) required for the osdc
  algorithm.

- [`simulate_registers()`](https://steno-aarhus.github.io/osdc/reference/simulate_registers.md)
  : Simulate a fake data frame of one or more Danish registers

- [`edge_cases()`](https://steno-aarhus.github.io/osdc/reference/edge_cases.md)
  : Create a synthetic dataset of edge case inputs

- [`non_cases()`](https://steno-aarhus.github.io/osdc/reference/non_cases.md)
  : List of non-cases to test the diabetes classification algorithm

- [`non_cases_metadata()`](https://steno-aarhus.github.io/osdc/reference/non_cases_metadata.md)
  :

  Description of the different non-cases included in
  [`non_cases()`](https://steno-aarhus.github.io/osdc/reference/non_cases.md)

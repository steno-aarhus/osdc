# Scrape SDS's website for their ICD-8 codes and saves to a file.

Scrape SDS's website for their ICD-8 codes and saves to a file.

## Usage

``` r
scrape_icd8_codes()
```

## Value

Saves a CSV file with ICD-8 codes. Outputs the path to the saved file.

## Examples

``` r
if (FALSE) { # \dontrun{
scrape_icd8_codes()
} # }
```

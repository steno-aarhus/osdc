# Create a vector of random NPU codes

Nomenclature for Properties and Units (NPUs) are codes that identifies
laboratory results.

## Usage

``` r
create_fake_npu(n)
```

## Arguments

- n:

  The number of NPUs to create.

## Value

A character vector.

## Examples

``` r
if (FALSE) { # \dontrun{
create_fake_npu(4)
} # }
```
